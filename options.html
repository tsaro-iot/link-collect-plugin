Link Collect Settings

Changes apply immediately to new scans.

URL Patterns

One pattern per line. Use * as wildcard (e.g. *://*.example.com/*)

Row Selector

CSS selector for the container block of each work order row. Used to associate a tracking number with its attachments.

Tracking Number Selector

CSS selector (within each row) for the element whose text is the tracking number. This will be prefixed to downloaded filenames.

Element Selector

CSS selector (within each row) for the element that holds the attachment links.

File Extensions

Comma-separated list of file extensions to look for.
