# link-fetcher-plugin
## Installation Guide

Welcome to the project! This is a Chrome plugin designed to enhance your browsing experience.

### How to Install

1. **Download the Source Code**
    - Clone or download this repository to your local machine.

2. **Open Chrome Extensions Page**
    - In Chrome, navigate to `chrome://extensions/`.

3. **Enable Developer Mode**
    - Toggle the "Developer mode" switch in the top right corner.

4. **Load Unpacked Extension**
    - Click the "Load unpacked" button.
    - Select the folder where you downloaded or extracted this project's source code.

5. **Enjoy!**
    - The extension should now be installed and active in your browser.

For any issues or questions, please refer to the project's README or open an issue on GitHub.
